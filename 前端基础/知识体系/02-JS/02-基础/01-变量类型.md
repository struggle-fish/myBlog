# 变量类型


JavaScript 中的变量类型分为两大类：基本类型（原始类型） 和 引用类型（对象类型）。



## 基本类型

基本类型是按值访问的简单数据段，存储在 **栈（Stack）** 中

- `Undefined`
- `Null`
- `Boolean`
- `Number`
- `String`
- `Symbol`



## 引用类型

引用类型是按引用访问的对象，存储在 **堆（Heap）** 中，赋值时传递的是内存地址（引用），修改会影响原对象。

- 普通对象 `let obj = { name: '张三' };`
- 数组 `let arr = [1, 2, 3];`
- 函数 `function fn() {}`
- 日期 `let date = new Date();`
- 正则 `let reg = /abc/;`
- Map/Set `let map = new Map(); let set = new Set();`
