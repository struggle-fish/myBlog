# 类型转化


## ==（抽象相等，面试最常考）


规则（按优先级从高到低）：

1. 特殊对：null == undefined → true（仅此一对，和其他值都不等）；

2. 数字 vs 字符串：字符串转数字，再比较；

3. 布尔值 vs 其他：布尔值转数字（true→1，false→0），再比较；

4. 对象 vs 原始值：对象转原始值（ToPrimitive），再比较；


```js

// 规则 1
null == undefined; // true
null == 0; // false（null 只和 undefined 相等）

// 规则 2
1 == '1'; // true（'1' → 1）
1 == '1a'; // false（'1a' → NaN）

// 规则 3
true == 1; // true（true → 1）
false == 0; // true（false → 0）
false == ''; // true（false → 0，'' → 0）
false == []; // true（false → 0，[] → '' → 0）

// 规则 4
[] == 0; // true（[] → '' → 0）
{} == '[object Object]'; // true（{} → '[object Object]'）

```



## ===（严格相等，不转换）

规则：只有类型和值都完全相等才返回 true，否则 false（没有任何隐式转换）。


```js
1 === '1'; // false（类型不同）
null === undefined; // false（类型不同）

```


::: tip

以下是 falsely 变量，除此之外都是 truly 变量

```js

!!0 === false

!!NaN === false

!!'' === false

!!null === false

!!undefined === false

!!false === false

```

:::
