# 重构阅读笔记


::: tip 什么是重构
所谓重构（refactoring）是这样一个过程：在不改变代码外在行为的前提下，对代码做出修改，以改进程序的内部结构

本质上说，重构就是在代码写好之后改进它的设计
:::
