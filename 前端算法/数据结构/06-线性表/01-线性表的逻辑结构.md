# 线性表的逻辑结构

## 定义

线性表（Linear List）是具有**相同数据类型**的n (n >= 0) 个数据元素的**有限序列**，其中 n 为表长，当 n = 0

时线性表是一个空表。若用 L 命名线性表，则其一般表示为：

`L = (a₁, a₂, ..., aᵢ, aᵢ₊₁, ..., aₙ)`

`aᵢ` 是线性表中的 “第i个”元素线性表中的位序


`a₁` 是表头元素， `aₙ` 是表尾元素。除第一个元素外，每个元素有且仅有一个直接前驱，除最后一个元素外，每个元素有且仅有一个直接后继。

## 基本操作

初始化线性表 （Initlist）。构造一个空的线性表 L ，分配内存空间

销毁操作（DestroyList），销毁线性表并释放线性表 L 所占用的内存空间。

插入操作（ListInsert），插入操作，在表 L 中的第 i 个位置上插入指定元素e

删除操作（ListDelete），删除操作，删除表 L 中第 i 个位置的元素，并用 e 返回删除元素的值。

按值查找（LocateElem），按值查找，在表 L 中查找具有给定关键字值的元素

按位查找（GetElem）按位查找，获取表 L 中第 i 个位置的元素的值。

求表长 （Length），返回线性表 L 的长度，即 L 中数据元素的个数。

输出操作（PrintList），输出操作，按照前后顺序输出线性表 L 的所有元素值

判空操作（Empty），判空操作，若 L 为空表则返回 true，否则返回 false
