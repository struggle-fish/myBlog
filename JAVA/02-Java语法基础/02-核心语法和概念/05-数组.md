# 数组

## 一维数组

### 概述

数组（Array）是一种用于存储多个相同类型数据的数据结构（可以理解为容器）。

- 一次性声明大量的用于存储数据的变量
- 要存储的数据通常都是同类型的数据，例如：考试成绩


```java 

int[] scores = {90, 80, 70, 60, 50};

```


### 定义格式
 
java 中的数组必须先初始化，然后才能使用。
所谓初始化就是为数组中的元素分配内存空间，并为每个元素赋值。

- 静态初始化：初始化时指定每个数组元素的初始值，由系统决定数组长度。适合一开始就能确定元素值的业务场景
- 动态初始化：初始化时只指定数组的长度，由系统为数组元素分配初始值。适合开始知道数据的数量，但是不确定具体元素值的业务场景。

::: code-group
```java [定义格式]

// 格式1 推荐使用
数据类型[] 数组名 

// 格式1的示例
int[] scores;
// 定义了一个int类型的数组，数组名是scores




// 格式2
数据类型 数组名[]

// 格式2的示例
int scores[];
// 定义了一个int类型的变量，变量名是source数组

```



```java [静态初始化]
// 格式
数据类型[] 数组名 = new 数据类型[]{元素1, 元素2, 元素3, ...};

int[] scores = new int[]{90, 80, 70, 60, 50};

// 简化格式
数据类型[] 数组名 = {元素1, 元素2, 元素3, ...};

int[] scores = {90, 80, 70, 60, 50};

```


```java [动态初始化]
// 格式
数据类型[] 数组名 = new 数据类型[数组长度];

int[] scores = new int[5];

// 动态初始化时，数组元素会被系统默认初始化值
// int类型的默认初始化值是0
System.out.println(scores[0]); // 0
```

:::


### 元素访问

- 数组内部保存的数据的访问方式：`数组名[索引]`

- 索引标识了元素在数组中的位置
- 索引用于访问数组中的数据使用，`数组名[索引]`等同于变量名，是一种特殊的变量名
- 数组的索引是从0开始的，连续的，每次增加1，直到数组的长度减1

```java 
int[] scores = {90, 80, 70, 60, 50};

System.out.println(scores);// 这里是数组的地址值

// 访问数组中的第一个元素
System.out.println(scores[0]); // 90
// 访问数组中的第三个元素
System.out.println(scores[2]); // 70

// 修改数组中的元素
scores[0] = 100;
System.out.println(scores[0]); // 100
```


### 常见操作

- 遍历数组：使用循环结构，依次访问数组中的每个元素

```java 
int[] scores = {90, 80, 70, 60, 50};

// 遍历数组
for(int i = 0; i < scores.length; i++) {
    System.out.println(scores[i]);
}
```


- 获取最大值：遍历数组，依次比较每个元素，记录最大值

```java 
int[] scores = {90, 80, 70, 60, 50};

// 获取最大值
int max = scores[0];
for(int i = 1; i < scores.length; i++) {
    if(scores[i] > max) {
        max = scores[i];
    }
}
System.out.println("最大值是：" + max);
```

- 数组打乱：遍历数组，随机交换每个元素的位置

```java 
int[] scores = {90, 80, 70, 60, 50};

// 数组打乱
Random random = new Random();
for(int i = 0; i < scores.length; i++) {
    // 生成一个随机索引，范围是[0, scores.length)，本质上就是索引的范围
    int randomIndex = random.nextInt(scores.length);
    
    // 交换当前元素和随机索引处的元素
    int temp = scores[i];
    scores[i] = scores[randomIndex];
    scores[randomIndex] = temp;
    
}

// 遍历数组
for(int i = 0; i < scores.length; i++) {
    System.out.println(scores[i]);
}
```

## 二维数组

### 概述

需求：存储多个班级的学生的考试成绩，改怎么办呢？

如果要存储一个班级中多个学生的考试成绩，就可以采用数组来存储。
多个班级的考试成绩，针对每个班级采用数组存储：

- 第一个班级： 数组1
- 第二个班级： 数组2
- 第三个班级： 数组3
- ...

但是多个班级也可以采用数组存储？所以 `java` 提供了二维数组。

### 定义格式

二维数组：元素为一堆数组的数组(存储一维数组的数组)

- 二维数组可以理解为一个表格，每个元素都是一个单元格，每个单元格可以存储一个值
- 二维数组的每个元素都是一个数组，每个数组的长度可以不同
- 二维数组的每个数组都有一个索引，用于访问数组中的元素


::: code-group
```java [定义格式]

// 格式1 推荐使用
数据类型[][] 数组名

// 格式1的示例
int[][] scores;
// 定义了一个int类型的二维数组，数组名是scores


// 格式2
数据类型 数组名[][]

// 格式2的示例
int scores[][];
// 定义了一个int类型的变量，变量名是source二维数组


// 格式3
数据类型[] 数组名[]

// 格式3的示例
int[] scores[];


```


```java [静态初始化]
// 格式
数据类型[][] 数组名 = new 数据类型[][]{
                        {元素1, 元素2, 元素3, ...}, 
                        {元素1, 元素2, 元素3, ...}, 
                        ...
                     };

// 示例
int[][] scores = new int[][]{{90, 80, 70}, {60, 50}};
// 定义了一个int类型的二维数组，数组名是scores，有两个元素，
// 每个元素都是一个int类型的数组，分别是{90, 80, 70}和{60, 50}

// 简化格式
数据类型[][] 数组名 = {
                {元素1, 元素2, 元素3, ...}, 
                {元素1, 元素2, 元素3, ...}, 
                ...
               };

// 示例
int[][] scores = {{90, 80, 70}, {60, 50}};
// 定义了一个int类型的二维数组，数组名是scores，有两个元素，
// 每个元素都是一个int类型的数组，分别是{90, 80, 70}和{60, 50}


```

```java [动态初始化]
// 格式
数据类型[][] 数组名 = new 数据类型[m][n];
m: 二维数组的长度(有多少个一维数组)
n: 二维数组中每个元素(一维数组)的长度(每一个一维数组的长度)


// 示例
int[][] scores = new int[3][3];
// 定义了一个int类型的二维数组，数组名是scores，有两个元素，
// 每个元素都是一个int类型的数组，长度都是3
```
::: 


### 元素访问

``` java

int[][] scores = new int[][]{{90, 80, 70}, {60, 50}};

// 格式
数组名[索引1][索引2]


// 访问数组中的第一个元素
System.out.println(scores[0][0]); // 90
// 访问数组中的第二个元素
System.out.println(scores[0][1]); // 80
// 访问数组中的第三个元素
System.out.println(scores[1][0]); // 60
```



### 常见操作

- 遍历数组：使用嵌套循环结构，依次访问数组中的每个元素

```java 
int[][] scores = {{90, 80, 70}, {60, 50}};
// 遍历数组
for(int i = 0; i < scores.length; i++) {
    for(int j = 0; j < scores[i].length; j++) {
        System.out.println(scores[i][j]);
    }
}

```


- 元素的打乱：遍历数组，随机交换每个元素的位置
```java 
int[][] scores = {{90, 80, 70}, {60, 50}};


 // 数组打乱
Random random = new Random();
for(int i = 0; i < scores.length; i++) {
    for(int j = 0; j < scores[i].length; j++) {
        // x 表示二维数组中的哪个数组
        int x = random.nextInt(scores.length);
        // y 表示这个一维数组中的哪一个位置
        int y = random.nextInt(scores[x].length);
        
        // 交换当前元素和随机索引处的元素
        int temp = scores[i][j];
        scores[i][j] = scores[x][y];
        scores[x][y] = temp;
    }
}
// 遍历数组
for(int i = 0; i < scores.length; i++) {
    for(int j = 0; j < scores[i].length; j++) {
        System.out.println(scores[i][j]);
    }
}
```
