# 循环


## for 循环

明确知道循环的次数

::: code-group

```java [格式]
for (初始化表达式; 循环条件; 迭代表达式) {
    // 循环体
}
```

```java [示例]
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
```

```java [示例2]
public class ForLoop {
    public static void main(String[] args) {
        int sum = 0
        for (int i = 0; i < 10; i++) {
            System.out.println(i);
            sum += i;
        }
        System.out.println("sum = " + sum);
    }
}
```

- 输出说有五位数的回文数

 ``12321, 10101`` 是回文数

```java [示例]
// 1. 获取五位数,通过循环实现
// 2. 获取每一个五位数的个位,十位,千位,万位
// 3. 判断是否是回文数
public class Palindrome {
    public static void main(String[] args) {
        for (int i = 10000; i < 100000; i++) {
            int ge = i % 10; // 个位
            int shi = (i / 10) % 10; // 十位
            int qian = (i / 1000) % 10; // 千位
            int wan = i / 10000 % 10; // 万位

            if (ge == wan && shi == qian) {
                System.out.println(i + " 是回文数");
            }
        }
    }
}
```







## while 循环

判断条件满足,就执行循环体

:::code-group

```java [格式]
初始化语句
while (循环条件) {
    // 循环体
    // 迭代表达式
}
```

```java [示例]
public class WhileLoop {
    public static void main(String[] args) {
        int i = 0;
        while (i < 10) {
            System.out.println(i);
            i++;
        }
    }
}
```

## do...while 循环

无论条件判断是否满足,至少执行一次循环体

:::code-group

```java [格式]
初始化语句
do {
    // 循环体
    控制语句
} while (循环条件);
```

```java [示例]
public class DoWhileLoop {
    public static void main(String[] args) {
        int i = 0;
        do {
            System.out.println(i);
            i++;
        } while (i < 10);
    }
}
```
