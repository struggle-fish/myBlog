# 函数传参


## 形参与实参介绍


形参（Formal Parameter）：在定义函数时，写在括号里的变量名称。它们是占位符，用于接收调用时传入的数据。

实参（Actual Parameter）：在调用函数时，实际传入的值（数据）。

形参 = 形式上的参数（变量名）

实参 = 实际传递的内容（具体值）

```py

def add(x, y):        # x 和 y 是【形参】
    return x + y

result = add(10, 20)  # 10 和 20 是【实参】

```


**形参和实参的关系**

- 在调用截断，实参（变量值）会绑定给形参（变量名）
- 这种绑定关系只能在函数体内使用
- 实参与形参的绑定关系再函数调用时生效，函数调用结束后会解除绑定关系
