# 函数的基本使用

## 什么是函数

函数就相当于具备某一个功能的工具，先定义后调用。

## 为何要用函数

不用函数的缺点：

- 组织结构不清晰，可读性差
- 代码冗余
- 可维护性，扩展性差




## 如何用函数


::: code-group

```python [格式]

def 函数名(参数1, 参数2, ...):
    """文档字符串（推荐写）"""
    # 函数体（要执行的代码）
    return 返回值   # 可选

```


```python [举例]

def add(a, b):
    """两个数相加并返回结果"""
    result = a + b
    return result

# 调用函数
sum1 = add(10, 25)
print(sum1)   # 35

```

:::


## 函数定义的三种形式


::: code-group


```py [无参函数]

def func():
  print('哈哈')
  print('哈哈')


# 定义函数发生的事情
# 1、申请内存空间保存函数体代码
# 2、将上述内存地址绑定函数名
# 3、定义函数不会执行函数体代码，但是会检测函数体语法


# 调用函数发生的事情
# 1、通过函数名找到函数的内存地址
# 2、然后加括号就是在触发函数体代码的执行 如 print()


```


```py [有参函数]

def func(x, y):
    print(x, y)


# 函数调用
func(1, 2)
```


```py [空函数]

def func(x, y):
  pass

```
:::




## 各种参数类型

```py

# 1. 位置参数（最常用）
def power(x, n):
    return x ** n

# 2. 默认参数（非常实用）
def greet(name, city="新加坡"):
    print(f"你好 {name}，欢迎来到 {city}")

greet("慢半拍")           # 使用默认值
greet("小红", "北京")     # 传入新值

# 3. 关键字参数（更清晰）
greet(name="张三", city="上海")

# 4. 可变参数 *args（元组）
def add_all(*numbers):
    return sum(numbers)

print(add_all(1, 2, 3, 4, 5))   # 15

# 5. 可变关键字参数 **kwargs（字典）
def student_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

student_info(name="慢半拍", age=25, city="新加坡")

```




## 函数的返回值


```py

# 返回单个值
def square(x):
    return x * x

# 返回多个值（本质是返回一个元组）
def get_info():
    return "慢半拍", 25, "新加坡"

name, age, city = get_info()   # 元组解包

```
