# 字符串类型


## 基础类


```py

s = "  Hello Python World!  "

print(len(s))                    # 长度 → 24
print(s.lower())                 # 全部转小写
print(s.upper())                 # 全部转大写
print(s.capitalize())            # 首字母大写
print(s.title())                 # 每个单词首字母大写
print(s.strip())                 # 去除两端空格（最常用！）
print(s.lstrip())                # 去除左边空格
print(s.rstrip())                # 去除右边空格

```


## 查找与判断类

```py

s = "hello python hello world"

print(s.find("python"))          # 返回第一次出现的位置 → 6（找不到返回 -1）
print(s.index("python"))         # 同 find，但找不到会报错
print(s.rfind("hello"))          # 从右边查找第一次出现的位置

print(s.count("hello"))          # 统计出现次数 → 2

print(s.startswith("hello"))     # 是否以 "hello" 开头 → True
print(s.endswith("world"))       # 是否以 "world" 结尾 → True

print("123".isdigit())           # 是否全是数字 → True
print("abc".isalpha())           # 是否全是字母 → True
print("abc123".isalnum())        # 是否只含字母和数字 → True
print("   ".isspace())           # 是否全是空白字符 → True

```


## 替换与拼接类

```py

s = "hello world hello python"

print(s.replace("hello", "hi"))           # 替换所有 "hello" → "hi world hi python"
print(s.replace("hello", "hi", 1))        # 只替换前1次

# 拼接（推荐方式）
words = ["Python", "is", "awesome"]
print(" ".join(words))                    # "Python is awesome"
print(",".join(words))                    # "Python,is,awesome"

# 字符串格式化（现代推荐写法）
name = "慢半拍"
age = 25
print(f"我是{name}，今年{age}岁")         # f-string（最推荐）
print("我是{}，今年{}岁".format(name, age))


```


## 分割与切片类

```py

s = "apple,banana,orange,grape"

print(s.split(","))              # 以逗号分割 → ['apple', 'banana', 'orange', 'grape']
print(s.split(",", 2))           # 只分割前2次

line = "name:张三 age:25 city:新加坡"
print(line.split())              # 默认按空白字符分割

# 路径分割示例
path = "C:/Users/慢半拍/Desktop/test.py"
print(path.split("/")[-1])       # 获取文件名 → test.py

```




## 字符串判断与清理


```py

# 清理用户输入
user_input = "  Hello   Python  "
clean = user_input.strip().lower()
print(clean)                     # "hello python"

# 判断是否为空
if not user_input.strip():       # 推荐写法
    print("输入不能为空")

# 去除所有空白字符（包括换行、制表符）
import re
clean_text = re.sub(r'\s+', '', user_input)   # 高级用法


```
