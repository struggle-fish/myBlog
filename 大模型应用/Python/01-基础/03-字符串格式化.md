# 字符格式化


Python 的格式化输出（字符串格式化）主要有 3 种 常用方式，从老到新依次是：

- `%` 格式化（最古老，Python 早期就存在）

- `str.format()` 方法（Python 2.6+ 引入，比较灵活）

- `f-string`（Formatted String Literals，Python 3.6+ 引入，目前最推荐）


## str.format() 方法

```py

name = "慢半拍"
age = 25
score = 98.765

# 位置参数
print("姓名：{}，年龄：{}，成绩：{:.2f}".format(name, age, score))

# 命名参数（更清晰）
print("姓名：{n}，年龄：{a}，成绩：{s:.2f}".format(n=name, a=age, s=score))

# 用字典展开（很方便）
info = {"name": "慢半拍", "age": 25}
print("你好，我是 {name}，今年 {age} 岁".format(**info))

```



## f-string 常用格式说明符（和 format() 通用）


在 {} 里用 : 后面加格式：

- {score:.2f} → 保留 2 位小数

- {num:,} → 千位分隔符（123,456）

- {num:>10} → 右对齐宽度 10

- {num:<10} → 左对齐

- {num:^10} → 居中

- {num:010} → 补 0 到宽度 10

- {name:>20} → 右对齐 20 个字符
