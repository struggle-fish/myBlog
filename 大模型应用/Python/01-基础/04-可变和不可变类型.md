# 可变和不可变类型


不可变类型（Immutable）：创建后 不能修改内容。如果你想“改”，Python 会创建一个新对象，原来的对象保持不变。

可变类型（Mutable）：创建后 可以直接修改内容（原地修改）。内存地址（id）不变。

可变类型：值改变，id不变，证明改的是原值，即原值是可以被改变的


![LOGO](/public/image/python/ScreenShot_2026-04-13_121240_961.png)

不可变类型： 

- int（整数）
- float（浮点数）
- str（字符串）
- bool（布尔）
- tuple（元组）
- None


可变类型：

- list（列表）
- dict（字典）
- set（集合）


```py
# 不可变类型：修改 = 创建新对象
a = 100
print(id(a))        # 假设地址是 1400000000

a = a + 50          # 不是修改，而是创建新整数对象
print(id(a))        # 地址变了！新的对象

# 可变类型：修改 = 原地改变
lst = [1, 2, 3]
print(id(lst))      # 假设地址是 2000000000

lst.append(4)       # 在原来的列表上直接修改
print(id(lst))      # 地址不变！还是同一个对象
print(lst)          # [1, 2, 3, 4]

```


**注意：多个变量指向同一个对象**

```py

# 不可变类型（安全）
x = 5
y = x
x = 10
print(y)        # 输出 5 （y 不受影响）

# 可变类型（危险！共享引用）
a = [1, 2, 3]
b = a           # b 和 a 指向同一个列表对象

a.append(999)
print(b)        # 输出 [1, 2, 3, 999]   ← b 也被改了！

```

```py
# 不会改外面的值：
def func(x):
    x = x + 1

a = 10
func(a)
print(a)  # 还是10

# 会直接改原对象：
def func(lst):
    lst.append(4)

a = [1,2,3]
func(a)
print(a)  # [1,2,3,4] 变了！

```
