# 文件操作

文件操作主要使用内置的 `open() `函数 + `with `语句（上下文管理器），强烈推荐使用 pathlib（现代、面向对象风格）来处理路径。


## 什么是文件

文件是操作系统提供给用户/应用程序操作硬盘的一种虚拟的接口



![LOGO](/public/image/python/ScreenShot_2026-04-21_111055_360.png)



为何要用文件：

- 用户/应用程序可以通过文件将数据永久保存在硬盘中，即操作文件就是操作硬盘

- 用户/应用程序直接操作的是文件，对文件的所有操作，都是在像操作系统发送调用，然后再由操作系统将其转化成具体的硬盘操作

如何用文件： `open()`




## open 功能的使用

- 控制文件读写内容的模式：t(文本) 和 b(二进制)，t和b必须跟r/w/a连用
  - t 文本 默认模式 ，读写都以 str 为单位

- 控制文件读写操作的模式：
  - `r` 只读
  - `w` 只写
  - `a` 追加写入
  - `+` : r+ 读写、w+读写覆盖、a+读写追加


- 常用组合：
  - `'r', encoding='utf-8'` → 读文本
  - `'w', encoding='utf-8'` → 写文本
  - `'rb' / 'wb' ` → 读写二进制

![LOGO](/public/image/python/ScreenShot_2026-04-21_113140_475.png)



## 文件操作流程

打开文件 -> 操作文件 -> 关闭文件


```py

from pathlib import Path

# 1. 读取文本文件（最常用）
file_path = Path("test.txt")

with open(file_path, "r", encoding="utf-8") as f:
    content = f.read()              # 读取全部内容
    # content = f.readline()        # 读取一行
    # content = f.readlines()       # 读取所有行，返回列表

print(content)

# 2. 写入文本文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello 慢半拍！\n")
    f.write("我在新加坡学习 Python。\n")

# 3. 追加内容
with open("log.txt", "a", encoding="utf-8") as f:
    f.write("2026-04-20 新记录\n")

```


**with** 上下文管理，with 语句，它会自动关闭文件，即使发生异常也不会忘记关闭

- 自动管理资源（打开 → 使用 → 关闭）
- 代码更简洁、安全
- 即使出现异常也能保证文件被关闭

```py

# 推荐写法（现代 Python 标准写法）
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# 执行完 with 代码块后，文件会自动关闭
print(f.closed)   # True

```



## 文件常用api

![LOGO](/public/image/python/ScreenShot_2026-04-21_125444_594.png)

- `read()` 读取全部内容 返回 `str / bytes`
- `readline()` 读取一行  返回 `str / bytes`
- `readlines()` 读取所有行 返回列表`list`
- `write(text)` 写入内容   返回写入字符数
- `writelines(lines)` 写入多行列表 返回`None`



## pathlib 常用方法

```py

from pathlib import Path

p = Path("folder/subfolder/file.txt")

print(p.exists())           # 文件/目录是否存在
print(p.is_file())          # 是否是文件
print(p.is_dir())           # 是否是目录
print(p.name)               # 文件名 → "file.txt"
print(p.stem)               # 无后缀文件名 → "file"
print(p.suffix)             # 后缀 → ".txt"
print(p.parent)             # 父目录
print(p.absolute())         # 绝对路径

# 创建目录
p.parent.mkdir(parents=True, exist_ok=True)

# 删除文件
p.unlink(missing_ok=True)

# 遍历目录
for file in Path(".").rglob("*.py"):   # 递归查找所有 .py 文件
    print(file)

```




## 总结

- 永远使用 `with open(...) as f:`
- 文本文件必须指定 `encoding="utf-8"`
- 读文件用 `'r'`，写文件用 `'w'`，追加用 `'a'`
- 二进制文件用 `'rb' / 'wb'`
- 大文件推荐逐行读取：`for line in f:`
- 路径管理推荐使用 `pathlib.Path`
