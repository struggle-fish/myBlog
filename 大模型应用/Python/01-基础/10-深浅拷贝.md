# 深浅拷贝



浅拷贝：只复制最外层对象，内层可变对象还是共享同一个引用（改一个会影响另一个）。


深拷贝：递归复制所有层级的对象，创建完全独立的新对象（内外都不互相影响）。


直接赋值 ≠ 拷贝，它只是让两个变量指向同一个对象。



![LOGO](/public/image/python/ScreenShot_2026-04-14_093922_086.png)



```py

import copy

# 原始数据（包含嵌套列表）
original = [1, 2, [3, 4, 5]]

# 1. 直接赋值（引用）
assign = original

# 2. 浅拷贝
shallow = copy.copy(original)          # 或者 original[:]  或 original.copy()
# shallow = list(original)             # 列表浅拷贝常用写法

# 3. 深拷贝
deep = copy.deepcopy(original)

print("修改前：")
print("original:", original)
print("shallow: ", shallow)
print("deep:    ", deep)

# 修改外层和内层
original[0] = 999          # 修改外层
original[2][0] = 888       # 修改内层可变对象

print("\n修改 original 后：")
print("original:", original)   # [999, 2, [888, 4, 5]]
print("shallow: ", shallow)    # [1, 2, [888, 4, 5]]   ← 内层被改了！
print("deep:    ", deep)       # [1, 2, [3, 4, 5]]     ← 完全不受影响


```



- 不可变类型（int/str/tuple）不存在深浅拷贝问题，也就是用原来的内存地址

- 只有 可变容器（list/dict/set）才会出现嵌套引用问题，重新复制一份新容器
