## 成员运算符

作用：判断一个值是否属于（存在于）另一个对象。

`in` 和 `not in`

支持的数据类型：

- 字符串（str）
- 列表（list）
- 元组（tuple）
- 字典（dict）—— 注意：默认判断 key
- 集合（set）
- 其他可迭代对象

```py

# 1. 字符串
name = "慢半拍"
print("慢" in name)        # True
print("半拍" in name)      # True
print("快" not in name)    # True

# 2. 列表 / 元组
fruits = ["苹果", "香蕉", "橙子"]
print("香蕉" in fruits)          # True
print("西瓜" not in fruits)      # True

# 3. 字典（默认判断 key）
person = {"name": "慢半拍", "age": 25, "city": "新加坡"}
print("name" in person)          # True
print("慢半拍" in person)        # False （因为不是 key）
print("age" in person)           # True

# 判断 value 是否存在（需要 values()）
print("慢半拍" in person.values())   # True

# 4. 集合
numbers = {1, 2, 3, 4}
print(5 not in numbers)          # True

```


## 身份运算符

`is` 和 `is not`

作用：判断两个变量是否指向同一个内存对象（即 id(a) == id(b)）


`==`：值是否相等

`is`：是不是同一个对象（身份相同）


```py

a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)      # True   ← 值相同
print(a is b)      # False  ← 不是同一个对象
print(a is c)      # True   ← 同一个对象

# 最常见的正确用法：判断 None
x = None
if x is None:          # 推荐写法
    print("x 是空值")

# 错误示范
if x == None:          # 也可以工作，但不推荐
    pass

```


```py

# 推荐写法
if name is None or name == "":
    print("名字为空")

if "admin" in user_roles:
    print("有管理员权限")

# 不推荐
if user == None:        # 应该用 is None
    pass
    
```
