# 集合类型

集合（set）是 Python 中一个非常实用但经常被低估的数据类型。

用来存放多个数据，自动去重

常用于：列表去重、判断成员关系、求交集 / 并集 / 差集

![LOGO](/public/image/python/ScreenShot_2026-04-20_132641_320.png)


与列表、字典的对比：

- 列表：有序、可重复、可变

- 字典：有序（3.7+）、键值对、可变

- 集合：无序、不重复、可变、元素必须是不可变类型（数字、字符串、元组）


## 定义


```py

# 正确定义
s1 = {1, 2, 3, 4, 5}
s2 = {"苹果", "香蕉", "橙子"}
s3 = set()                    # 空集合！注意不能用 {} 

# 错误示范
empty = {}                    # 这其实是空字典！不是空集合

# 从其他类型转换
lst = [1, 2, 2, 3, 3, 4]
s4 = set(lst)                 # 自动去重 → {1, 2, 3, 4}

```


## 增删改查


::: code-group

```py [增]

s = {1, 2, 3}

s.add(4)                      # 添加单个元素
s.update([5, 6, 7])           # 添加多个元素（可迭代对象）
s.update("abc")               # 添加字符串的每个字符

```


```py [删]

s = {1, 2, 3, 4, 5}

s.remove(3)                   # 删除指定元素，不存在会报错 KeyError
s.discard(10)                 # 删除元素，不存在也不会报错（推荐）
s.pop()                       # 随机删除并返回一个元素（因为无序）
s.clear()                     # 清空整个集合

```

```py [查]
s = {"苹果", "香蕉", "橙子"}

print("香蕉" in s)            # True
print("西瓜" not in s)        # True

```
:::


## 集合运算

```py

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)          # 并集（union）          → {1,2,3,4,5,6}
print(a & b)          # 交集（intersection）   → {3,4}
print(a - b)          # 差集（difference）     → {1,2}
print(a ^ b)          # 对称差集               → {1,2,5,6}

# 方法形式（更清晰）
print(a.union(b))
print(a.intersection(b))
print(a.difference(b))
print(a.symmetric_difference(b))

```


## 子集、超集

```py

a = {1, 2, 3}
b = {1, 2, 3, 4, 5}

print(a.issubset(b))      # a 是 b 的子集？ → True
print(b.issuperset(a))    # b 是 a 的超集？ → True
print(a.isdisjoint(b))    # 两个集合没有交集？ → False

```
