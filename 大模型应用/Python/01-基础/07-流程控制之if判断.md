# 流程控制之if判断


`if` 是 `Python` 中最重要、最常用的流程控制语句，用来让程序根据条件决定要不要执行某段代码。


## 基本语法


```py

# 写法1：最简单 if
age = 20
if age >= 18:
    print("你已经成年了")

# 写法2：if - else（二选一）
if age >= 18:
    print("成年")
else:
    print("未成年")

# 写法3：if - elif - else（多条件判断，最常用）
score = 85

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")



```
注意📢：

- 条件后面必须有冒号 :
- 属于 if 的代码块必须统一缩进（推荐使用 4 个空格）
- elif 可以写很多个，else 只能有一个且放在最后


## 链式比较

python 独有的写法

```py

age = 25

if 18 <= age <= 30:          # 等价于 age >= 18 and age <= 30
    print("你是青年")

if 0 < score < 100:
    print("分数有效")

```


## 常用条件写法


```py

# 1. 比较运算
if age > 18:
if score >= 60:
if name != "":

# 2. 逻辑运算组合
if age >= 18 and score >= 60:
if age < 18 or money == 0:
if not is_student:      # 推荐写法

# 3. 成员运算
if "admin" in roles:
if key in my_dict:

# 4. 身份运算（最常用）
if user is None:
if result is not None:

# 5. 真值判断（简洁写法）
if name:          # name 不为空字符串
if my_list:       # 列表不为空
if data:          # data 不为 None、0、空等

```

```py

# 推荐写法示例
username = input("请输入用户名：").strip()

if not username:
    print("用户名不能为空！")
elif len(username) < 3:
    print("用户名长度不能少于3个字符")
elif " " in username:
    print("用户名不能包含空格")
else:
    print(f"欢迎你，{username}！")

```
