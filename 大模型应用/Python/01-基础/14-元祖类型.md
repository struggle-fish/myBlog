# 元祖类型

元组（tuple）是 Python 中非常重要的不可变序列类型。

![LOGO](/public/image/python/ScreenShot_2026-04-17_134032_022.png)




## 定义


```py

# 常见定义方式
t1 = (1, 2, 3, 4, 5)
t2 = ("苹果", "香蕉", "橙子")
t3 = ()                    # 空元组
t4 = (10,)                 # 单个元素的元组，必须加逗号！（重点）
t5 = 1, 2, 3               # 小括号可以省略（Python 特色）

print(type(t1))            # <class 'tuple'>
print(type(t4))            # <class 'tuple'>

```


## 常用操作


```py
t = ("慢半拍", 25, "新加坡", "学习", "游戏", 25)

# 1. 基础信息
print(len(t))                    # 长度 → 6
print(t[0])                      # 索引取值 → "慢半拍"
print(t[-1])                     # 倒数第一个 → 25

# 2. 切片（和列表完全一样）
print(t[1:4])                    # (25, '新加坡', '学习')
print(t[::-1])                   # 反转元组

# 3. 查找类
print(t.index("新加坡"))         # 返回第一次出现的位置 → 2
print(t.count(25))               # 统计出现次数 → 2

# 4. 成员判断
print("学习" in t)               # True
print("唱歌" not in t)           # True


```


## 解包

```py

# 基本解包
name, age, city = ("慢半拍", 25, "新加坡")
print(name, age, city)

# 常用技巧：函数返回多个值
def get_user_info():
    return "小红", 20, "北京", "女"

name, age, city, gender = get_user_info()

# 忽略某些值（用 _ 占位）
name, _, city, _ = get_user_info()
print(name, city)

# * 收集剩余元素
first, *middle, last = (1, 2, 3, 4, 5, 6)
print(first)   # 1
print(middle)  # [2, 3, 4, 5]
print(last)    # 6


```



## 元祖和列表转换

```py

lst = [1, 2, 3, 4]

t = tuple(lst)           # 列表 → 元组
new_lst = list(t)        # 元组 → 列表

```
