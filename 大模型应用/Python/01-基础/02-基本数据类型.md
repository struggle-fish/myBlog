# 基本数据类型


Python 一共有 8 种 常用的基本数据类型，按实际使用频率从高到低排序如下：

::: code-group

```py [数值类型]

# int（整数） - 任意大小，Python 自动处理大整数
a = 100
b = -999999999999999
c = 2 ** 100   # 很大也没问题

# float（浮点数）
x = 3.14159
y = 2.0        # 带小数点就是 float
z = 1e10       # 科学计数法 = 10000000000.0

# 类型转换
print(int(3.9))     # 3 （向下取整）
print(float(5))     # 5.0

```


```py [字符串]

name = "慢半拍"
city = '新加坡'
info = """我是多行
字符串"""

# 常用操作
print(len(name))           # 长度
print(name + " 你好")      # 拼接
print(name * 3)            # 重复
print("半" in name)        # 成员判断

# f-string（Python 3.6+ 强烈推荐）
age = 25
print(f"我叫{name}，今年{age}岁，来自{city}")

```

```py [布尔]
is_student = True
is_adult = False

print(5 > 3)      # True
print(bool(0))    # False
print(bool(""))   # False
print(bool([]))   # False

# 所有会被判定为 False 的值：0, 0.0, "", None, [], {}, ()
```

```py [列表]
# （可变、有序）
fruits = ["苹果", "香蕉", "橙子"]

fruits.append("葡萄")      # 添加
fruits.insert(1, "芒果")   # 插入
fruits.remove("香蕉")      # 删除
print(fruits[0])          # 取值
print(len(fruits))

```

```py [元祖]
# 不可变，有序

colors = ("红色", "绿色", "蓝色")
# colors[0] = "黑色"   # 会报错！不可修改

# 常用场景：函数返回多个值
def get_info():
    return "张三", 18, "男"

name, age, gender = get_info()   # 元组解包

```

```py [字典]
person = {
  "name": "慢半拍",
  "age": 25,
  "city": "新加坡",
  "hobbies": ["游戏", "旅行"]
}

print(person["name"])
print(person.get("age"))       # 推荐写法，更安全

person["age"] = 26             # 修改
person["country"] = "新加坡"    # 新增

```

```py [集合]
# 集合set 可变、无序、不重复

a = {1, 2, 3, 3, 3}     # 自动去重 → {1, 2, 3}
b = {3, 4, 5}

print(a | b)    # 并集
print(a & b)    # 交集
print(a - b)    # 差集

```
