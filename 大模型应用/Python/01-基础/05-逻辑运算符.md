# 逻辑运算符

## 示例

![LOGO](/public/image/python/ScreenShot_2026-04-13_131506_374.png)


`not` 把紧跟其后的那个条件结果取反


```py
a = True
b = False
c = 5
d = 0

# 1. and（都为真才为真）
print(a and b)        # False
print(c > 0 and d == 0)   # True

# 2. or（只要有一个为真就为真）
print(a or b)         # True
print(c < 0 or d == 0)    # True

# 3. not（取反）
print(not a)          # False
print(not d)          # True （因为 0 被视为 False）

```

```py
# not > and > or 优先级

3 > 4 and not 4 > 3 or 1 == 3 and 'x' == 'x' or 3 > 3
#     False                   False                   False
(3 > 4 and (not 4 > 3)) or (1 == 3 and 'x' == 'x') or 3 > 3

```

## 短路运算


```py

# and：前面为 False，后面的表达式不会执行（短路）
print(False and 1/0)     # 不会报错，直接返回 False

# or：前面为 True，后面的表达式不会执行（短路）
print(True or 1/0)       # 不会报错，直接返回 True

```


## 真值和假值规则

以下内容当作 False

- False
- 0、0.0
- ''（空字符串）
- []、()、{}（空列表、元组、字典）
- None

其他所有情况都被视为 True。


```py

print(bool(""))      # False
print(bool([]))      # False
print(bool("hello")) # True
print(bool([1,2]))   # True

```
