# 字典类型

字典是 Python 中最重要的数据结构之一，用于存储 键值对（key-value） 数据，类似现实中的“电话簿”或 JSON 格式。



## 基本概念

- 可变（mutable）
- 无序（Python 3.7+ 保持插入顺序）
- 键（key）必须是不可变类型（str、int、tuple 等）
- 键唯一，值可以重复
- 查找速度极快（哈希表实现）

```py

# 定义字典
empty = {}                          # 空字典
person = {
    "name": "慢半拍",
    "age": 25,
    "city": "新加坡",
    "is_student": False,
    "hobbies": ["学习", "旅行", "摄影"]
}

print(type(person))   # <class 'dict'>
print(len(person))    # 键的数量 → 5

```

## 增 / 改

```py

person = {"name": "慢半拍", "age": 25}

# 最常用方式
person["city"] = "新加坡"           # 添加新键值对
person["age"] = 26                  # 修改已有键的值

# 使用 update()
person.update({"gender": "男", "job": "程序员"})
person.update(age=27, score=95)     # 也可以这样写

```


## 查


```py

person = {"name": "慢半拍", "age": 26, "city": "新加坡"}

# 推荐写法（安全，不会报错）
print(person.get("name"))           # "慢半拍"
print(person.get("phone"))          # None（键不存在时返回 None）
print(person.get("phone", "未知"))  # 设置默认值 → "未知"

# 直接取值（键不存在会报 KeyError）
print(person["name"])

# 同时获取 key 和 value（最常用遍历方式）
for key, value in person.items():
    print(f"{key}: {value}")

```



## 删

```py

person = {"name": "慢半拍", "age": 26, "city": "新加坡", "score": 95}

person.pop("score")                 # 删除指定 key，并返回 value
person.pop("phone", None)           # 键不存在也不会报错

del person["city"]                  # del 方式删除

person.clear()                      # 清空整个字典


```




## 其他


```py

person = {"name": "慢半拍", "age": 26, "city": "新加坡"}

print(person.keys())                # 获取所有键 → dict_keys(['name', 'age', 'city'])
print(person.values())              # 获取所有值 → dict_values(...)
print(person.items())               # 获取所有键值对 → dict_items([...])

# 合并字典（Python 3.9+ 新语法，推荐）
d1 = {"a": 1, "b": 2}
d2 = {"b": 3, "c": 4}
merged = d1 | d2                    # {'a': 1, 'b': 3, 'c': 4}

```
